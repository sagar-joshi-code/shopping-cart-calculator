
Description

This is a simple JavaScript project that calculates the total price of products in a shopping cart, applies a discount, and displays the final price.
It helps practice arrays, loops, and conditional logic.

Features

Stores product prices in an array

Counts total items

Calculates total price

Applies 10% discount

Displays final price after discount


Example Output
Total items = 5
script.js:12 Total price = 1820
script.js:15 Final price after discount = 1638
